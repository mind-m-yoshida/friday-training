friday-training
===============

## 準備

* クローンする
	* `$ git clone https://github.com/mind-m-yoshida/friday-training.git`
	* `$ cd friday-training`
* ダウンロードする
	* *Download ZIP*ボタンでダウンロードした後、解凍する。

## セットアップ

### Gradleのセットアップ

`$ gradlew.bat`を実行する。

### eclipseのプロジェクト作成

`$ eclipse-project-setup.bat`を実行する。

### eclipseへのインポート

* File -> Importへ移動。
* General -> Existing Project into Workspaceを選択。
* Select root directoryで上記のプロジェクトを選択。
* Finishで完了。
